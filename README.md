##安装软件
```
1. windows 按组合键 WIN+R 键输入CMD进入命令行
* 切换盘符: > D:

2. 进入项目解压目录
* 进入目录: > cd ShowQuantitative

3. 启动 Docker Desktop

4. 安装软件
* 安装软件: > docker-compose up -d --build
```

##相关操作
```
1. 浏览器打开网页 http://127.0.0.1:8080/#/
2. 浏览器打开网页 http://127.0.0.1:9527/ 没有报404说明安装成功
3. 软件启停操作通过Docker Desktop操作即可
```
